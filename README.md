# CAPA Management System

Sistem manajemen Corrective Action & Preventive Action untuk maintenance mesin produksi.

## ✨ Fitur
- Login multi-role (Admin & User)
- Dashboard dengan statistik CAPA
- Input data CAPA baru
- Update progress CAPA
- Filter data (Line, Mesin, Bulan)
- Export PDF & CSV
- Dark/Light mode
- Manage Users (Admin only)

## 🛠️ Teknologi
- HTML5, CSS3, JavaScript
- Chart.js untuk visualisasi
- jsPDF & SheetJS untuk export
- MySQL database
- PHP backend API
